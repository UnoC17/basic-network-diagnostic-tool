# Basic Network Diagnostic Tool
This Python script pings a list of IP addresses and reports connectivity status and response time.

## Features
- Checks if IP addresses are online or offline
- Displays response time
- Simple, beginner-friendly automation
- Great for networking practice

## How to Run
1. Download `network_diagnostic.py`
2. Open Terminal
3. Navigate to the file
4. Run `python3 network_diagnostic.py`
